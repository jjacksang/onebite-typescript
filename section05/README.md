## 타입스크립트 인터페이스에 대하여

~~인터페이스란~~

-   타입에 이름을 지어주는 또 다른 문법
-   객체의 구조를 정의하는데 특화된 문법(상속, 합침 등의 특수한 기능을 제공함)

-   인터페이스를 동일한 이름으로 작성하여도 최종적으로 합쳐서 적용
    -- 단, 이미 작성해준 타입을 재정의는 불가능
